# 🐱 Cat's Cafe WPF

A desktop café billing application developed in **C#** using **Windows Presentation Foundation (WPF)**.

## 📖 Description

**Cat's Cafe WPF** is a desktop application that allows users to create and manage café bills through a graphical user interface.

The application enables users to:

* Add menu items
* Remove items from the bill
* Calculate GST (5%)
* Add a tip (percentage or fixed amount)
* Display the final bill
* Save bills to a file
* Load bills from a file
* Clear the current bill

The project demonstrates the use of WPF controls, event-driven programming, file handling, input validation, and object-oriented programming principles. WPF provides a modern graphical interface using XAML for UI design and C# for application logic.

## 📋 Application Functions

* **Add Item** – Adds a new item to the bill.
* **Remove Item** – Removes the selected item.
* **Add Tip** – Adds either a percentage-based or fixed tip.
* **Display Bill** – Calculates and displays:

  * Net Total
  * Tip
  * GST (5%)
  * Total Amount
* **Save** – Saves the current bill to a file.
* **Load** – Loads a saved bill.
* **Clear** – Removes all current bill items.
